Description
TermSuggest est une application console .NET 8 qui implémente un algorithme de suggestion de termes similaires basé sur la similarité textuelle.

Installation
Prérequis
.NET 8 SDK

IDE recommandé : Visual Studio 2022

Utilisation
Mode Démonstration
L'application lance automatiquement des démonstrations

Projet de Démarrage :

Application console TermSuggest.Cli
